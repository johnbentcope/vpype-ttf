# vpype-ttf
Vpype plugin for ttf
See: https://github.com/abey79/vpype

Requires freetype-py

* Install repository:
* `pip install .` while in the repository directory.

Have font file specified, eg:
* `vpype render ./Vera.ttf "Vpype!" write a.svg`
